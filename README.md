# 🛠️ Hibernate CRUD Application

A simple Java-based CRUD (Create, Read, Update, Delete) application using **Hibernate ORM** and **MySQL** as the backend database. This project demonstrates how to perform basic database operations using Hibernate with proper configurations and annotations.

---

## 📌 Features

- Add new records to the database (Create)
- Retrieve all or specific records (Read)
- Modify existing records (Update)
- Delete records from the database (Delete)
- Clean architecture with DAO and Service layers
- Uses Hibernate’s annotation-based configuration

---

## 🧰 Tech Stack

- **Java**
- **Hibernate ORM**
- **MySQL**
- **Maven (for dependency management)**
- **JDBC**
- **Log4j (for logging)**
